
ReadMe For: CyberSecruityHelpChatBotPoe
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
This is a basic chat bot that uses a dictionary to store a pre-set number of key values that is used to compare to user input to give a semi related response.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Overview of functions.

1.Plays a greeting audio on launch
2.Uses voice synthesizer to turn all text from questions into speech
3.Displays a asci logo with an asci cyber on launch and 3 sec goes away
4.Asks user for there name and outputs a hello message based on their input
5.Prompts user for a input to search with or add a task
6.Gives a result based on input, in the form of asnwered question of added task.
7.Asks and then stores a reminder date if a task was added.
8.Users are given a random response from a preset of responses based on the found key word of phrase.
9.If it is users second or more question users will get a definition of a previous question as a fun fact.
10.Allows users to see and delete tasks in a task panel
11.Allows users to see activity log on the activity panel
12.A quiz is avaible on a quiz panel where uses select a combo box button to choose their answer.
13.Display if users answer was correct with a reason as to why the correct answer is correct.
14.keeps track of the quiz score and gives a message at the end based on the results of the quiz
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
How to set up CyberSecruityHelpChatBotPoe and run it.
Installation:
0.have all software related such as windows installed.
1.Download all associate files on GitHub.
2.Unzip the folder.
3.Open the folder and click on the file called CyberSecruityHelpChatBotPoe then click on the CyberSecruityHelpChatBotPoe.csproj to launch the code.
4.Use visual studio when launching.


How to run:
1.In visual studio click the start button to start running the code.
2.Console will launch, wait for audio to finish playing
3.Enter your name
4.Enter add task followed by task name to add a task or enter a question to get a answer.
5.Click the panel called quiz to begin the quiz
6.Press the button that has the value you belive to be correct and press the enter button.
7.press the next question and repeate step 6-7 untill quiz is done
8.click the tasks panel to see all added tasks
9.Click on value and press delete button to delete a value
10.click on the log tab to see a log of activities.
